# CAD & 3D Design Portfolio

A single-page CAD and 3D design portfolio. Dark technical aesthetic, built with vanilla HTML/CSS/JS. No frameworks or dependencies (except Google Fonts).

## Features

- Dark technical/blueprint aesthetic with dark sage green accent
- Animated skill proficiency bars
- Scroll reveal animations
- Project detail modals
- Cross-linked to graphic design portfolio
- Responsive layout

## Structure

```
index.html   ← rename cad-index.html to this before uploading
README.md
```

## GitHub Pages Deployment

1. **Rename** `cad-index.html` → `index.html` before uploading
2. Push to a new GitHub repo (e.g. `cad-portfolio`)
3. Go to **Settings → Pages → Source: main branch, root folder**
4. Live at `https://yourusername.github.io/cad-portfolio`

## Cross-Linking Both Sites

Once both repos are live, update the links in each file:

**In this file (CAD portfolio):** find and replace both instances of:
```
https://yourusername.github.io/portfolio
```
with your actual graphic design portfolio URL.

**In your graphic design `index.html`:** add a link to this site in the footer or nav.
